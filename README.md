# Freewrite — Unfiltered Thinking, Persistent Memory

A distraction-free writing environment that removes the crutch of editing — and slowly builds a personal AI context layer from everything you've written.

## Vision

Most writing tools make it too easy to second-guess yourself. Freewrite removes the escape hatch: no backspace, no edits, no pauses. You write forward — always forward — for a fixed amount of time.

## Core Experience

- **The Session**: Type until the timer ends. No backspace, no menus, no distractions.
- **The Context Layer**: Every session is saved as a structured record, enriched by AI to understand your patterns, themes, and focus areas.
- **AI Companion**: Query your own archive to surface insights, generate tasks, and reflect on your writing.
